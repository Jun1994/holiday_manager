# 假期管理系统

一个基于Python和PyQt6开发的Windows桌面应用程序，用于管理加班、调休、年假及特殊假期。

## 功能特性

### 1. 加班管理
- 记录加班日期、时间段和备注
- 自动计算等效假期天数
  - 周末加班：满6小时=1天，不满6小时=0.5天
  - 工作日晚上加班：统一0.5天
- 显示加班记录列表，包含已用天数和剩余天数
- 剩余天数颜色标识（红色、绿色、灰色）

### 2. 调休管理
- 使用加班产生的调休进行休假
- 支持拆分使用（最小单位0.5天）
- 自动校验可用天数
- 过期机制：每年指定日期自动作废

### 3. 年假管理
- 根据社保缴费年限设置天数
- 支持拆分使用
- 可设置过期日期

### 4. 特殊假期管理
- 支持自定义假期类别（婚假、丧假、病假等）
- 每个类别可独立设置名称、天数、过期日期和显示颜色
- 支持编辑和删除自定义类别

### 5. 假期总览
- 主界面顶部显示各类假期的使用情况
- 卡片式布局，不同类别使用不同颜色
- 显示总计、已用、剩余天数

### 6. 日历视图
- 支持月视图和年视图切换
- 不同颜色标识不同类型的活动
- 点击日期查看详细信息
- 图例说明

### 7. 休假申请
- 统一的休假申请入口
- 支持全天或指定时间段
- 提供上午/下午快捷按钮
- 自动校验可用天数

### 8. 系统设置
- 假期类别管理（添加、编辑、删除）
- 调休过期日期设置
- 系统类别保护（年假、调休不可删除）

## 技术架构

- **编程语言**: Python 3.x
- **界面框架**: PyQt6
- **数据库**: SQLite
- **打包工具**: PyInstaller（可选）

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 使用说明

### 首次使用

1. 运行程序
2. 进入"系统设置"标签页
3. 点击"打开设置"按钮
4. 在"假期类别"标签中设置年假、婚假、丧假等假期的总天数
5. 在"系统设置"标签中设置调休过期日期（可选）
6. 开始使用

### 录入加班

1. 进入"加班管理"标签页
2. 点击"添加加班"按钮
3. 填写加班日期、开始时间、结束时间
4. 系统自动计算等效假期天数
5. 可选填写备注说明
6. 点击"保存"

### 申请休假

1. 进入"休假申请"标签页
2. 点击"申请休假"按钮
3. 选择假期类型
4. 如果是调休，选择加班日期
5. 选择休假日期
6. 选择时间段（可选）
7. 填写休假天数
8. 可选填写备注说明
9. 点击"提交"

### 查看日历

1. 进入"日历视图"标签页
2. 可切换"月视图"/"年视图"
3. 使用"上月"/"下月"或"上年"/"下年"按钮切换
4. 休假和加班日期会在日历上以不同颜色显示
5. 点击日期查看详细信息

### 管理假期类别

1. 进入"系统设置"标签页
2. 点击"打开设置"按钮
3. 在"假期类别"标签中：
   - 点击"添加假期类别"新增
   - 点击"编辑"修改现有类别
   - 点击"删除"删除自定义类别

## 数据存储

- 数据库文件：`holiday_manager.db`
- 位置：程序运行目录

## 注意事项

1. **系统类别保护**：
   - "年假"和"调休"是系统类别，不能删除
   - 只能编辑这两个类别的天数和过期设置

2. **过期提醒**：
   - 程序启动时会自动检查过期的调休
   - 过期的调休会自动作废并显示提示

3. **拆分使用**：
   - 调休、年假、特殊假期都支持拆分使用
   - 最小单位为0.5天

4. **时间段使用**：
   - 休假可以只选择日期（全天）
   - 也可以选择具体时间段（上午/下午）
   - 选择时间段会自动设置为0.5天

## 打包为可执行文件（可选）

使用PyInstaller打包为Windows可执行文件：

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "假期管理系统" main.py
```

打包后的可执行文件位于 `dist` 目录下。
